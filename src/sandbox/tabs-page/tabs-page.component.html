Tabs Components

Allows you switching between views without navigating to another route. Consists of two components: tabs wrapper and tab itself.

Tabs Input/ Output

@Output tabChanges T Emits instance of rendered component

Tab Input/ Output

@Input componentRef Function Reference to a component to render
@Input selected Boolean If applied, renders selected tab initially
