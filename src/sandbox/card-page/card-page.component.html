Card Component

Styling component for blocks with separated logic

Directives

os-card[widget] default styling: space theme on a background, max-width: 280px
os-card[dark] default styling: dark violet background

Input/ Output

@Input divided boolean Layout changed to two columns
@Input error boolean For default card sets visual feedback of error state
@Input openBordersOnMobile boolean Removes borders on mobile, splits divided card into to separate blocks
