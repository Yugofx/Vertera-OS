Button Component

Button is a main UI component. All the native functionality was left as is, except some extensions. Can be applied to buttons and anchor elements.

Directives

os-button default styling: no shadows, no outline
os-button-raised default styling: box shadow is attached, no outline
os-button-stroke default styling: outline is attached, no box shadow

Input/ Output

@Input disabled boolean Sets interactive state of a button
@Input color 'accent' | 'basic' Sets background color of a button

Examples

  • Config: [os-button]
  • Config: [os-button-raised], color="accent"
  • Config: [os-button-stroke], color="basic"
