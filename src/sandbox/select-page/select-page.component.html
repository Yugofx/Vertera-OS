Select Components

Imitates work of a native html select. Consists of two components: select itself and options containing values and displayed values

Select Input/ Output

@Input value String Default value can be set here
@Input disabled Boolean Sets select disable state
@Output valueChange String Emits value of selected option
@Output openChange - Emits event on select list opening
@Output closeChange - Emits event on select list closing

Option Input/ Output

@Input value String Value, that will be emitted on option select
@Input viewOption String Defines a visible value of an option
@Input selected boolean Sets selected by default option

Examples

  • State: {{ select1.value }}
  • State: Disabled
  • State: {{ select2.value }}
