Toggle Components

Group of elements (switcher, checkbox) for user interaction.
Has two states: true and false.
Can be used as a part of Angular form module's control.

Input/ Output

@Input id String Sets id to angular component
@Input required Boolean Used with angular forms, sets required validator
@Input name String Sets name attribute to component
@Input value String Sets value attribute to component
@Input labelPosition 'before' | 'after' Sets label position before/after checkbox
@Output change Boolean Emits true or false on user click
